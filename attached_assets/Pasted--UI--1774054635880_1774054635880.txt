선생님모드 스케줄러를 월간 중심의 통합 업무 스케줄러로 전면 개편해줘.
이번 작업은 단순 달력 UI 수정이 아니라, 스케줄러를 출결/일지/보강/메모의 중심 허브로 만드는 작업이다.

[1. 기본 구조]

1. 스케줄러 기본 진입 화면은 무조건 `월간 스케줄`로 설정
2. 상단 보기 탭은 `월 / 주 / 일`만 유지
3. 첫 진입 시 항상 `월` 탭 선택 상태
4. 월간 달력 전체가 보이게 구성
5. 오늘 날짜는 강조 표시

[2. 월간 달력 날짜 칸 표시 방식]

1. 날짜 칸에는 반 이름을 직접 표시하지 말 것
2. 대신 그 날짜에 수업이 있는 `시간`만 표시
예:
- 15시
- 16시
- 17시

3. 한 날짜에 수업이 여러 개면 시간순으로 위에서 아래 또는 짧은 줄 형태로 표시
4. 너무 많으면 일부만 보이고 나머지는 점/더보기 느낌으로 처리 가능하지만, 기본은 시간 표시 우선

[3. 날짜 클릭 동작]

1. 월간 달력에서 날짜 칸을 누르면 해당 날짜 상세 팝업(또는 바텀시트)이 열리게 해줘
2. 날짜 클릭은 토글 방식으로 동작
- 날짜 클릭 → 상세 팝업 열림
- 같은 날짜 다시 클릭 → 상세 팝업 닫힘
- 팝업 바깥 빈 화면 클릭 → 상세 팝업 닫힘

3. 팝업이 닫히면 다시 월간 달력 원화면만 보이게 해줘

[4. 날짜 상세 팝업 구조]

팝업에는 아래 내용을 포함

- 선택한 날짜
- 그 날짜 수업 리스트
- 각 수업의 시간 / 반명 / 인원 수
- 날짜 메모
- 작업 버튼

예:
15:00 초급반 / 5명
16:00 중급반 / 4명
17:00 개인레슨 / 1명

[5. 수업 리스트 정렬 규칙]

1. 날짜 팝업 안 수업 리스트는 항상 `시간순`으로 정렬
2. 위에서부터 아래로 빠른 시간 → 늦은 시간 순서
3. 완료/미완료를 따로 그룹으로 나누지 말 것

[6. 완료된 수업 표시 방식]

1. 완료된 수업은 별도 배지보다 `글자 전체 취소선(가로줄)`으로 표시
2. 예:
- 15:00 초급반 → 수업이 끝났으면 전체 텍스트에 가로줄
- 16:00 중급반 → 아직 안 끝났으면 일반 표시

3. 완료된 스케줄이 무엇인지 직관적으로 보이게 할 것
4. 항상 시간순 정렬은 유지하고, 완료된 것만 글자에 줄 긋는 방식으로 구분

[7. 반 카드 클릭 동작]

1. 날짜 상세 팝업 안에서 반 카드를 누르면 학생 리스트를 팝업 안에 직접 펼치지 말 것
2. 반 상세 페이지로 이동하게 해줘
3. 반 상세 페이지에서 가능한 작업
- 학생 리스트 확인
- 출석 체크
- 결석 처리
- 반 조정
- 적용 버튼

4. `적용` 또는 `저장`을 누르면 홈으로 가지 말고 반드시 방금 작업하던 `같은 날짜 상세 팝업 상태`로 복귀
예:
- 26일 팝업에서 15:00반 클릭
- 반 상세 페이지 이동
- 적용
- 다시 26일 상세 팝업 복귀

[8. 보강 기능]

1. 날짜 상세 팝업 안에 `보강` 버튼 또는 탭 추가
2. 누르면 기존 보강 관리 리스트 페이지로 이동
3. 보강 작업 후 `적용 완료` 누르면 다시 같은 날짜 상세 팝업으로 복귀
4. 홈으로 보내면 안 됨

[9. 일지 작성 기능]

1. 날짜 상세 팝업 안에 `일지 작성` 버튼 추가
2. 누르면 기존 일지 작성 페이지로 이동
3. 일지 작성 완료 후 다시 같은 날짜 상세 팝업으로 복귀
4. 복귀 후 해당 수업 카드가 즉시 `완료 상태`로 갱신되게 해줘
5. 일지 작성은 팝업 안에서 긴 입력을 직접 하지 말고 별도 페이지 이동 방식 유지

[10. 메모 기능]

1. 메모는 공용 메모/반별 메모로 나누지 말고 `날짜 메모` 하나만 사용
2. 날짜 상세 팝업 안에서 날짜 메모 확인/수정 가능
3. 달력 날짜 칸에는 메모 유무만 표시 가능
4. 날짜 단위 메모 예:
- 학부모 요청
- 행사
- 특이사항
- 준비물

[11. 우측 하단 + 버튼]

기능은 아래로 구성
- 수업 추가
- 보강 추가
- 개인레슨 추가
- 메모 추가

[12. 필터 기능]

필터는 유지/추가
- 내 반만 보기
- 특정 반 선택
- 특정 시간대
- 보강만 보기
- 미작성 일지만 보기

[13. 우선순위/깃발 기능]

깃발 기능은 유지하되 우리 용도에 맞게 사용
- 긴급
- 체크 필요
- 일반
- 없음

[14. 일정 없음 화면]

날짜에 수업이 없으면 아래처럼 표시
- `이 날은 수업이 없습니다`
- 보강 추가 버튼
- 메모 추가 버튼

[15. 가장 중요한 복귀 규칙]

반 상세 / 보강 / 일지 작성 어떤 작업을 하더라도
- 절대 홈으로 보내지 말 것
- 절대 스케줄러 첫 화면으로만 초기화하지 말 것
- 반드시 방금 작업하던 `같은 날짜 상세 팝업 상태`로 복귀시킬 것

즉 흐름은 항상 아래여야 함
월간 달력
→ 날짜 클릭
→ 날짜 상세 팝업
→ 반 상세/보강/일지 작성 이동
→ 적용/완료
→ 다시 같은 날짜 상세 팝업 복귀

[16. 작업 후 직접 확인]

아래 항목 실제 눌러서 확인
1. 월간 스케줄 기본 진입 확인
2. 날짜 칸에 반 이름이 아니라 수업 시간이 표시되는지 확인
3. 날짜 클릭 시 팝업 열림
4. 같은 날짜 다시 클릭 시 팝업 닫힘
5. 빈 화면 클릭 시 팝업 닫힘
6. 팝업 안 수업 리스트가 시간순인지 확인
7. 완료된 수업은 글자 전체에 취소선이 들어가는지 확인
8. 반 상세 이동 후 적용 → 같은 날짜 팝업 복귀 확인
9. 보강 이동 후 적용 완료 → 같은 날짜 팝업 복귀 확인
10. 일지 작성 후 완료 → 같은 날짜 팝업 복귀 확인
11. 날짜 메모 저장/수정 확인
