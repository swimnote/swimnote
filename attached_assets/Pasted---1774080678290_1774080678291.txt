관리자모드 스케줄러를 부분 수정하지 말고,
선생님모드 스케줄러 구조를 기준으로 아예 동일하게 복사 적용해서 통일해줘.

지금 문제:
- 선생님모드 스케줄러는 일간 탭 구조가 잘 되어 있음
- 관리자모드 스케줄러는 구조가 다르고 엉망임
- 이미 두 번 수정 요청했는데도 선생님모드와 동일하게 안 바뀜

이번에는 "비슷하게"가 아니라
`선생님모드 스케줄러 UI/동작/탭 순서/상세 흐름을 관리자모드에 그대로 이식`해야 한다.

==================================================
[1. 최우선 목표]
==================================================

관리자모드 스케줄러를
선생님모드 스케줄러와 아래 항목이 전부 동일하게 맞춰줘.

동일해야 하는 것:
1. 상단 탭 구조
2. 월/주/일 탭 순서
3. 선택 상태 UI
4. 날짜 이동 방식
5. 상세 진입 방식
6. 카드 스타일
7. 버튼 배치
8. 뒤로가기 흐름
9. 홈 버튼 동작
10. 팝업/페이지 복귀 방식

즉,
관리자모드 스케줄러는 새로 따로 만들지 말고
선생님모드 스케줄러를 관리자용 데이터만 넣어서 재사용하는 방식으로 바꿔줘.

==================================================
[2. 탭 순서 통일]
==================================================

관리자모드 스케줄러 상단 탭 순서를
선생님모드와 완전히 똑같이 맞춰줘.

순서:
- 월
- 주
- 일

지금처럼 관리자모드에서 순서 다르거나 구조 다르면 안 됨.
선생님모드 스케줄러의 탭 순서와 동일하게 고정해줘.

==================================================
[3. 일간 탭 관리자모드 표시 방식]
==================================================

관리자모드는 여러 선생님/여러 반 수업 정보가 들어오기 때문에
일간 탭은 리스트형 표시가 맞다.

단, 이 리스트형도
선생님모드 일간 탭 UI를 기준으로 확장해서
관리자용 데이터만 더 보여주는 방식으로 정리해줘.

관리자모드 일간 탭 리스트 표시 항목 예시:
- 시간
- 반명
- 담당 선생님
- 인원 수
- 상태 요약

즉,
기본 레이아웃과 인터페이스는 선생님모드와 동일하게 유지하고
관리자모드에서는 "담당 선생님 정보"만 추가로 보이게 해줘.

==================================================
[4. 선생님모드 스케줄러 기능을 관리자모드에 그대로 이식]
==================================================

관리자모드에도 선생님모드에 있는 아래 기능을 똑같이 옮겨줘.

1. 월간 스케줄 화면
- 날짜 칸 표시 방식
- 일정 시간 표시 방식
- 오늘 강조
- 선택한 날짜 표시

2. 날짜 선택 시 상세 진입 방식
- 관리자모드도 동일한 흐름 사용

3. 일간 화면 구조
- 요일 선택 방식
- 리스트 카드 스타일
- 반 카드 클릭 동작

4. 뒤로가기/복귀 흐름
- 선생님모드와 같은 방식으로 적용

즉,
관리자모드는 스케줄러만 따로 다르게 만들지 말고
선생님모드 스케줄러를 기반 컴포넌트처럼 재사용해줘.

==================================================
[5. 관리자모드에서만 추가되는 차이]
==================================================

관리자모드는 선생님모드와 동일한 스케줄러를 쓰되
데이터 범위만 다르게 한다.

차이점:
1. 여러 선생님 수업이 함께 보임
2. 일간 리스트에서 담당 선생님 이름 표시
3. 필요 시 필터로 특정 선생님/특정 반만 볼 수 있게 구조 열어두기

즉,
"관리자모드는 데이터만 넓고 UI는 선생님모드와 동일"
이 원칙으로 가줘.

==================================================
[6. 강력한 구현 방식 요구]
==================================================

이번에는 관리자모드 스케줄러를 따로 만지작거리지 말고,
선생님모드 스케줄러 컴포넌트/구조를 기준으로 복사 또는 공용화해줘.

가능하면:
- 공통 SchedulerLayout
- 역할별 데이터만 주입
- teacher mode = 본인 반 데이터
- admin mode = 전체 반/전체 선생님 데이터

즉,
관리자 스케줄러를 별도 디자인으로 유지하지 말고
선생님 스케줄러의 공용화/재사용으로 해결해줘.

==================================================
[7. 반드시 직접 확인]
==================================================

이번 작업은 꼭 실제로 확인하고 보고해줘.
대충 바뀐 것처럼 말하지 말고 아래를 직접 체크해줘.

1. 선생님모드 스케줄러 진입
2. 월/주/일 탭 순서 확인
3. 일간 탭 UI 확인

4. 관리자모드 스케줄러 진입
5. 월/주/일 탭 순서가 선생님모드와 동일한지 확인
6. 일간 탭 구조가 선생님모드와 동일한지 확인
7. 관리자모드에서는 담당 선생님 정보가 추가로 보이는지 확인
8. 월간/주간/일간에서 공통 레이아웃이 통일됐는지 확인

==================================================
[8. 절대 금지]
==================================================

1. 관리자모드를 따로 대충 비슷하게 만들기 금지
2. 탭 순서 다르게 두기 금지
3. 월/주/일 UI 스타일 다르게 두기 금지
4. 뒤로가기 흐름 다르게 두기 금지
5. 선생님모드에서 이미 잘된 구조를 관리자모드에서 다시 깨는 것 금지

==================================================
[9. 최종 목표]
==================================================

최종 목표는 이거다.

- 선생님모드 스케줄러 = 기준형
- 관리자모드 스케줄러 = 기준형을 그대로 사용 + 전체 데이터/담당선생님 정보만 추가

즉 사용자가 보면
"선생님모드와 관리자모드 스케줄러가 같은 시스템처럼 느껴져야 한다"

==================================================
[10. 보고 형식]
==================================================

작업 후 아래 형식으로 보고해줘.

- 관리자모드 월/주/일 탭 순서: 통일됨 / 미해결
- 관리자모드 일간 탭 UI: 선생님모드와 동일 / 미해결
- 관리자모드 월간 탭 UI: 선생님모드와 동일 / 미해결
- 관리자모드 주간 탭 UI: 선생님모드와 동일 / 미해결
- 관리자모드에서 담당 선생님 정보 표시: 통과 / 미해결
- 공통 스케줄러 구조 재사용 여부: 예 / 아니오

문제가 남아 있으면
어느 탭이 아직 다르고
왜 다른지 정확히 적어줘.
