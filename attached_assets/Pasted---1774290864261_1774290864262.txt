[수영장 관리자 모드 메뉴 중복 정리 지시서 - 수업설정/운영설정 역할 분리 + 휴무일 관리 위치 정리]

지금부터 수영장 관리자 모드에서
수업설정과 운영설정 사이에 중복되어 있는 “수영장 설정/수업운영설정” 구조를 정리한다.

이번 작업 목표는 아래 2가지다.

1. 수업설정 안에 있는 `수업운영설정` 항목 제거
2. 운영설정에만 `수영장 설정`을 남기고,
   그 수영장 설정 페이지 제일 아래에 있는 `휴무일 관리` 버튼 제거

중요:
- 메뉴 이름만 지우는 게 아니라 연결 구조와 중복 개념까지 정리해야 한다
- 수업설정은 “수업 관련 세부 정책”
- 운영설정은 “수영장 전체 운영/기본 환경”
이 역할이 겹치지 않게 정리해야 한다
- 이미 휴무일 관리는 매출관리 아이콘 항목 쪽에 있으므로 수영장 설정 안에서는 제거한다
- 실제 클릭 테스트 후 완료 보고할 것

==================================================
1. 최종 구조 기준
==================================================

최종 기준은 아래처럼 고정한다.

[수업설정]
- 보강정책 설정
- 권한 설정
- 알림 설정
- 레벨/테스트 설정
- 피드백 기본설정

수업설정에서 제거:
- 수업운영설정 ❌

--------------------------------------------------

[운영설정]
- 수영장 설정 ⭕

운영설정 안의 수영장 설정은
수영장 기본 정보 / 운영 기본 환경 / 공통 설정 역할만 담당

==================================================
2. 수업설정에서 제거할 항목
==================================================

제거 대상:
- 수업운영설정

확인할 것:
- 수업설정 팝업/바텀시트/메뉴 배열에서 해당 아이콘 제거
- 눌렀을 때 연결되던 route/action 제거
- 빈 자리 어색하지 않게 정렬 재배치
- 다른 버튼 위치가 무너지지 않게 정리

중요:
- UI만 숨기지 말고 메뉴 매핑/클릭 핸들러/라우팅도 정리
- 수업운영설정 관련 잘못 남은 연결이 없는지 확인

==================================================
3. 운영설정에만 수영장 설정 유지
==================================================

운영설정 안에는 `수영장 설정`만 남긴다.

즉,
수영장의 기본 정보/공통 운영 관련 설정은
운영설정 → 수영장 설정
이 한 곳에서만 관리되게 한다.

중요:
- 같은 개념의 설정이 수업설정과 운영설정에 중복되면 안 된다
- 관리자 입장에서 헷갈리지 않게 역할을 명확히 분리할 것

==================================================
4. 수영장 설정 페이지에서 제거할 항목
==================================================

제거 대상:
- 제일 아래 `휴무일 관리` 버튼

이유:
- 휴무일 관리는 이미 매출관리 아이콘 항목에 들어가 있음
- 수영장 설정 안에 또 있으면 중복이라 혼란 발생

해야 할 것:
- 수영장 설정 페이지 하단의 휴무일 관리 버튼 제거
- 해당 버튼이 연결하던 route/action도 같이 정리
- 수영장 설정 페이지 레이아웃이 비어 보이지 않게 간격 정리

==================================================
5. 역할 구분을 명확히 할 것
==================================================

이번 정리 후 역할은 이렇게 고정:

수업설정:
- 수업 운영에 직접 영향을 주는 정책/설정
- 보강/알림/권한/레벨/피드백

운영설정:
- 수영장 기본 환경/공통 설정
- 수영장 설정

매출관리 또는 별도 위치:
- 휴무일 관리

중요:
이 역할을 코드/메뉴/UX에서 일관되게 유지할 것

==================================================
6. 함께 점검할 것
==================================================

이번 수정하면서 아래도 같이 점검:
- 수업설정 팝업 안에 더 겹치는 항목 없는지
- 운영설정 안에 수업설정 성격의 항목이 섞여 있지 않은지
- 휴무일 관리가 실제로 매출관리 쪽에서 정상 연결되는지
- 수영장 설정에서 휴무일 버튼 제거 후 다른 기능 안 깨지는지

==================================================
7. 테스트 필수 시나리오
==================================================

반드시 실제 클릭 테스트:

시나리오 A
- 관리자 홈
- 수업설정 열기
기대 결과:
- 수업운영설정 항목이 더 이상 보이지 않음

시나리오 B
- 관리자 홈
- 운영설정 진입
- 수영장 설정 진입
기대 결과:
- 수영장 설정은 정상 열림
- 하단에 휴무일 관리 버튼 없음

시나리오 C
- 관리자 홈
- 매출관리(또는 현재 휴무일 관리가 위치한 아이콘) 진입
기대 결과:
- 휴무일 관리 기능은 그 위치에서 정상 사용 가능

시나리오 D
- 수업설정 안의 나머지 항목 클릭
기대 결과:
- 보강정책 설정 / 권한 설정 / 알림 설정 / 레벨/테스트 설정 / 피드백 기본설정 정상 연결
- 메뉴 배열 뒤틀림 없음

==================================================
8. 수정 금지
==================================================

절대 깨지면 안 되는 것:
- 보강정책 설정 기능
- 권한 설정 기능
- 알림 설정 기능
- 레벨/테스트 설정 기능
- 피드백 기본설정 기능
- 운영설정의 다른 기본 기능
- 매출관리 쪽 휴무일 관리 기능

이번 작업은
중복 메뉴 정리와 위치 정돈 작업이다.

==================================================
9. 완료 보고 형식
==================================================

반드시 아래 형식으로 보고하라.

1. 수업설정에서 제거한 항목
2. 운영설정에 남긴 수영장 설정 구조
3. 수영장 설정에서 제거한 휴무일 관리 버튼 처리 방식
4. 수정한 파일 목록
5. 수업설정 UI/연결 테스트 결과
6. 운영설정/수영장 설정 테스트 결과
7. 매출관리 쪽 휴무일 관리 연결 테스트 결과
8. 남아 있는 중복 항목 여부

중요:
- “삭제 완료” 한 줄 보고 금지
- 실제 눌러보고 연결 확인한 결과를 적어라
- 미확인 항목은 미확인이라고 적어라
