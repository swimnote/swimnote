관리자 모드의 “매출관리” 탭 안에 “휴무일 지정” 기능을 추가하라.

이번 작업의 목표는
관리자가 휴무일을 미리 지정하고,
그 휴무일을 기준으로 선생님 스케줄표와 다음 달 수업 발생에 반영하여
5주차 관리와 휴무일 관리를 일관되게 처리하는 것이다.

==================================================
[1. 기능 위치]

관리자 모드 > 매출관리 탭 안에
월 표시 영역(예: 2026년 03월 정산 / 월 선택 영역) 옆에
“휴무일 지정” 버튼을 추가한다.

즉:
- 월 선택
- 휴무일 지정

이 두 요소가 매출관리 상단에서 함께 보이게 할 것

==================================================
[2. 휴무일 지정 기능 목표]

관리자가 특정 날짜를 휴무일로 지정하면,
그 날짜는 선생님들의 수업 스케줄표에서 “휴무일”로 표시되고,
해당 날짜에는 일반 수업이 자동 발생되지 않도록 연결한다.

핵심 목적:
- 5주차 관리
- 휴무일 반영
- 수업 발생 차단
- 실제 운영 달력 기준 정산 관리

==================================================
[3. 휴무일 지정 화면/팝업 구성]

휴무일 지정 버튼을 누르면
월 이동이 가능한 달력 선택 화면 또는 팝업이 열리게 한다.

기능 요구사항:
1. 월 이동 가능
2. 다음 달만 아니라 몇 달 뒤도 지정 가능
3. 특정 날짜 직접 선택 가능
4. 요일 단위 일괄 지정 가능
5. 지정된 휴무일은 달력에서 표시
6. 해제도 가능

==================================================
[4. 기본 옵션]

휴무일 지정 화면에는 아래 기본 옵션을 제공한다.

기본 체크 ON:
- 공휴일
- 일요일

즉, 휴무일 지정 화면을 열면
기본적으로 “공휴일 자동 휴무”와 “일요일 휴무”가 체크된 상태여야 한다.

관리자는 필요 시 아래 동작 가능:
- 공휴일 체크 해제
- 일요일 체크 해제
- 특정 공휴일 날짜만 수업일로 다시 지정
- 특정 일요일만 수업일로 다시 지정

핵심:
기본은 편하게 자동 체크
예외는 수동 조정 가능하게

==================================================
[5. 요일 지정 기능]

달력에서 날짜 개별 지정 외에,
요일별 일괄 지정 기능을 제공한다.

예:
- 월요일 전체 휴무
- 화요일 전체 휴무
- 일요일 전체 휴무

해당 월 기준으로 요일별 일괄 지정되게 하고,
여러 요일 중복 선택 가능하게 할 것

==================================================
[6. 선생님 스케줄표 연결]

관리자가 휴무일을 지정하면,
선생님 스케줄표에 해당 날짜가 아래처럼 반영되어야 한다.

1. 해당 날짜 셀에 “휴무일” 표시
2. 일반 수업 생성 불가
3. 기존 수업이 이미 있으면 경고 또는 별도 표시
4. 휴무일 날짜는 일반 날짜와 시각적으로 구분

즉,
관리자 설정 → 선생님 스케줄표 반영
이 연결이 반드시 살아 있어야 한다

==================================================
[7. 다음 달 수업 발생 연결]

휴무일은 다음 달 수업 발생 로직과 반드시 연결한다.

규칙:
1. 휴무일로 지정된 날짜는 수업 발생 대상에서 제외
2. 공휴일/일요일 자동 체크 상태도 반영
3. 휴무일 때문에 빠진 회차는
   이후 미실시/보강 흐름으로 처리 가능한 구조를 유지

핵심:
5주차를 억지 계산하는 게 아니라
휴무일을 제외한 실제 수업 가능일 기준으로 수업 발생 수를 계산하게 한다

==================================================
[8. 저장 방식]

휴무일 지정은 월별/날짜별로 저장되게 한다.

요구사항:
1. 특정 월의 휴무일 데이터 저장
2. 다음 달뿐 아니라 미래 월도 저장 가능
3. 다시 열면 기존 지정 상태 유지
4. 공휴일/일요일 자동 체크 결과도 반영 상태 유지

==================================================
[9. UI/사용성]

사용하기 편하게 아래를 반영하라.

1. 휴무일 지정 버튼은 매출관리 상단에서 잘 보이게 배치
2. 달력에서 선택된 날짜는 색상/라벨로 명확히 표시
3. 공휴일/일요일 자동 체크 옵션은 상단에 배치
4. 날짜 개별 선택 + 요일 일괄 선택 둘 다 가능하게
5. 저장 버튼 명확히 제공
6. 닫고 다시 들어와도 상태가 유지되게

==================================================
[10. 기존 로직 보호]

중요:
- 기존 정산 로직은 최대한 유지
- 기존 수업/출결/보강 데이터 구조는 함부로 변경하지 말 것
- 이번 작업은 휴무일 지정 기능 추가 및 연결 작업이다
- 기능 삭제 금지
- 기존 버튼/연결 끊기 금지

==================================================
[11. 테스트 지시]

수정 후 반드시 실제 테스트하라.

테스트 1.
관리자 매출관리 상단에 “휴무일 지정” 버튼이 보이는지 확인

테스트 2.
휴무일 지정 버튼 클릭 시 월 이동 가능한 달력 화면/팝업이 열리는지 확인

테스트 3.
특정 날짜를 휴무일로 지정하고 저장하면 다시 열었을 때 유지되는지 확인

테스트 4.
공휴일 / 일요일이 기본 체크 ON 상태인지 확인

테스트 5.
공휴일 체크 해제 후 특정 공휴일을 수업일로 다시 지정 가능 여부 확인

테스트 6.
요일 일괄 지정(예: 일요일 전체 휴무)이 정상 작동하는지 확인

테스트 7.
관리자에서 지정한 휴무일이 선생님 스케줄표에 “휴무일”로 표시되는지 확인

테스트 8.
휴무일 날짜에는 일반 수업 발생 또는 생성이 막히는지 확인

테스트 9.
다음 달뿐 아니라 몇 달 뒤 월에도 휴무일 지정이 가능한지 확인
