지금부터 관리자 모드와 선생님 모드에 저장공간/클라우드 사용량 기능을 추가하라.

이번 작업 목표는
업로드 데이터 과금 기준이 되는
“계정별 저장공간 사용량”과 “수영장 전체 총 사용량”을
실제로 계산하고 화면에 표시하는 것이다.

중요:
- 단순 사진/영상 개수 표시가 아니라 실제 사용 용량 총합이어야 한다
- 데이터는 해당 계정이 사용하는 모든 저장 데이터 총합이다
- 메신저, 일지, 첨부, 공지, 시스템 기본 저장분까지 포함해야 한다
- 관리자에게는 전체 총합과 선생님별 사용량이 보여야 한다
- 선생님에게는 본인 개인 사용량이 보여야 한다
- 게이지/합계가 실제로 맞는지 반드시 확인할 것

==================================================
[1. 최종 목표]

관리자 모드:
- 더보기 안 적절한 위치에 “저장공간 관리” 또는 “클라우드 사용량” 섹션 추가
- 수영장 전체 총 사용량 표시
- 선생님별 사용량 표시
- 항목별 총합 표시 가능

선생님 모드:
- 더보기 또는 설정 안에 “내 저장공간” 섹션 추가
- 본인 총 사용량 표시
- 카테고리별 사용량 표시

==================================================
[2. 포함해야 하는 데이터 범위]

이번 사용량 계산은
단순 사진/영상만이 아니라
그 계정이 사용하는 모든 데이터 총합으로 계산하라.

반드시 포함:
- 사진 업로드 데이터
- 영상 업로드 데이터
- 메신저 텍스트 데이터
- 메신저 첨부파일 데이터
- 수영일지 본문 데이터
- 수영일지 첨부 데이터
- 공지사항 본문/첨부 데이터
- 기타 계정이 생성한 저장 데이터
- 기본 시스템 저장분(계정/프로필/필수 기본 구조 등)도 포함

중요:
“그 계정이 사용하는 모든 용량 총합” 기준으로 계산할 것

==================================================
[3. 관리자 모드 UI]

관리자 > 더보기 안에 아래 구조 추가

섹션명 추천:
- 저장공간 관리
또는
- 클라우드 사용량

표시 구조:
1. 전체 사용량 요약 카드
   - 현재 총 사용량
   - 총 제공 용량
   - 남은 용량
   - 사용률 퍼센트
   - 게이지 바

예:
- 3.2GB / 5GB 사용 중
- 64%

2. 카테고리별 총합
   - 사진
   - 영상
   - 메신저
   - 수영일지
   - 공지/문서
   - 기타

3. 선생님별 사용량 리스트
   각 선생님별 표시:
   - 이름
   - 총 사용량
   - 사용률 또는 전체 대비 비중
   - 상세보기 가능하면 좋음

==================================================
[4. 선생님 모드 UI]

선생님 > 더보기 또는 설정 안에 아래 구조 추가

섹션명:
- 내 저장공간

표시 구조:
1. 내 총 사용량 요약 카드
   - 현재 사용량
   - 제공 용량
   - 남은 용량
   - 사용률 퍼센트
   - 게이지 바

2. 카테고리별 사용량
   - 사진
   - 영상
   - 메신저
   - 수영일지
   - 공지/문서
   - 기타

3. 안내 문구
   - 업로드한 파일과 메신저, 일지 등 계정이 사용하는 전체 저장공간입니다.

==================================================
[5. 표시 단위]

표시 단위는 가장 익숙한 형태로 자동 변환하라.
예:
- B
- KB
- MB
- GB

예를 들어 매우 작은 데이터는 B/KB,
일반 사용량은 MB,
큰 사용량은 GB로 자동 변환

==================================================
[6. 게이지 바 규칙]

관리자/선생님 모두 게이지 바를 넣어라.

표시 방식:
- 사용량 / 총량 숫자 병기
- 가로 프로그레스 바

색상 추천:
- 0~69% 기본색
- 70~89% 경고색
- 90% 이상 위험색

중요:
게이지 값은 실제 계산된 총합과 일치해야 한다.

==================================================
[7. 관리자 총합 계산 규칙]

관리자 총 사용량은
해당 수영장에 소속된 모든 계정 데이터 합산으로 계산하라.

최소 포함:
- 관리자 본인 데이터
- 부관리자 데이터
- 선생님 데이터
(학부모까지 포함할지 여부는 현재 과금 구조에 맞춰 판단하되, 우선 선생님 중심이면 선생님/직원 기준으로 구현 가능)

중요:
선생님별 사용량을 모두 더한 값과
전체 총합 카드 수치가 논리적으로 맞아야 한다.

==================================================
[8. 선생님별 리스트 상세]

관리자 화면의 선생님별 사용량 리스트는
익숙한 저장공간 UI처럼 보이게 한다.

예:
- 김선생   1.2GB
- 박코치   840MB
- 이코치   120MB

가능하면 클릭 시 상세 팝업:
- 사진
- 영상
- 메신저
- 일지
- 기타
- 총합

최소 구현:
리스트에 총합 사용량 표시

==================================================
[9. 현재 화면과 연결]

지금 이미 사진/영상 사용량 비슷한 표시가 일부 있다면
그걸 버리지 말고 확장해서 재사용하라.

단,
반드시 아래로 확장되어야 한다:
- 사진/영상만이 아니라 전체 저장공간 총합
- 관리자 = 전체 + 선생님별
- 선생님 = 내 총합

==================================================
[10. 데이터 계산 방식]

각 데이터 항목의 실제 저장 크기를 기준으로 계산하라.

메신저/일지처럼 텍스트 데이터는
문자열 byte length 또는 현재 저장 구조 기준으로 계산하고,
첨부가 있으면 첨부 파일 크기도 포함

기본 시스템 저장분도 반영하라.
예:
- 계정 기본 레코드
- 프로필 정보
- 필수 구조 데이터

중요:
대충 개수 기반으로 계산하지 말고,
가능한 한 실제 저장 크기 합산 기준으로 계산할 것

==================================================
[11. 기능 연결 체크 필수]

이번 작업은 반드시 실제 연결 확인을 하라.

체크 항목:
1. 선생님이 사진 업로드 시 내 저장공간 수치 증가하는지
2. 선생님이 메신저/일지 작성 시 수치 증가하는지
3. 관리자 전체 총합에도 반영되는지
4. 선생님별 리스트 합계와 전체 총합이 맞는지
5. 게이지 퍼센트 계산이 숫자와 일치하는지
6. 단위 변환(B/KB/MB/GB)이 자연스러운지

==================================================
[12. 테스트 시나리오]

반드시 실제 테스트하라.

시나리오 A.
선생님 계정으로 사진 업로드
→ 선생님 내 저장공간 증가 확인
→ 관리자 총합 증가 확인

시나리오 B.
선생님 계정으로 메신저 메시지 작성
→ 선생님 내 저장공간 증가 확인
→ 관리자 총합 증가 확인

시나리오 C.
선생님 계정으로 수영일지 작성
→ 선생님 내 저장공간 증가 확인
→ 관리자 총합 증가 확인

시나리오 D.
선생님별 사용량 리스트의 합계와
관리자 총 사용량 카드가 일관되는지 확인

시나리오 E.
게이지 퍼센트와 실제 사용량/총량 계산이 맞는지 확인

==================================================
[13. 파일 수정 전 먼저 확인]

작업 전에 아래를 먼저 확인하라.

1. 현재 사진/영상 사용량 표시 관련 파일
2. 업로드 데이터 저장 구조 파일
3. 메신저 데이터 저장 파일
4. 수영일지 저장 파일
5. 공지/문서 저장 파일
6. 관리자 더보기 화면 파일
7. 선생님 더보기/설정 화면 파일

==================================================
[14. 작업 후 보고 형식]

작업 후 아래 순서로 보고하라.

1. 수정 파일 목록
2. 관리자 화면에 추가한 저장공간 구조
3. 선생님 화면에 추가한 저장공간 구조
4. 어떤 데이터 항목을 합산했는지
5. 총합 계산 방식
6. 게이지 계산 방식
7. 테스트 시나리오 A~E 결과
8. 아직 후속으로 필요한 항목이 있으면 명시

==================================================
[15. 절대 금지]

- 사진/영상 개수만 보여주고 총량 계산 안 하는 것 금지
- 가짜 게이지 바 금지
- 선생님 총합과 관리자 총합이 안 맞는 상태 금지
- 텍스트 데이터(메신저/일지) 제외 금지
- 화면만 만들고 실제 반영 안 되는 상태 금지

이번 작업은
관리자는 전체 저장공간 총합과 선생님별 사용량을 보고,
선생님은 본인 개인 사용량을 보는 구조를
실제로 동작하게 만드는 작업이다.
