일지 작성 페이지와 학부모/선생님 앨범 구조를 아래 기준으로 전면 보완해줘.
이번 작업의 핵심은 `일지에 사진/영상 업로드 기능 추가`와 `업로드된 미디어가 선생님/학부모 화면에서 올바르게 저장·노출되게 하는 것`이다.

[1. 일지 작성 페이지 업로드 기능 추가]

현재 일지 작성 페이지에 사진 업로드 기능이 없으니 아래 업로드 버튼을 추가해줘.

1. 반 공통 일지 영역 아래에 추가
- 반 전체 사진 업로드
- 반 전체 영상 업로드

2. 학생별 추가 일지 영역마다 각각 추가
- 학생 사진 업로드
- 학생 영상 업로드

즉 구조는 아래처럼 가야 함

반 공통 일지
- 본문 입력
- 반 전체 사진 업로드 버튼
- 반 전체 영상 업로드 버튼

학생별 추가 일지
- 학생별 본문 입력
- 학생 사진 업로드 버튼
- 학생 영상 업로드 버튼

[2. 업로드 후 일지 작성 페이지 안에서 보여주는 방식]

1. 업로드하면 해당 영역 아래에 미디어가 쭉 나열되게 해줘
2. 표시 순서는 `업로드한 순서대로`
3. 세로 스크롤로 계속 내려보는 방식
4. 사진/영상 섞여 있을 수 있음
5. 일지 작성 중에도 업로드한 항목이 바로 미리보기로 보이게 해줘

[3. 선생님 쪽 저장 규칙]

업로드된 파일은 아래 경로 기준으로 저장/분류해줘.

1. 반 전체 사진 업로드
→ 선생님 앨범의 `반 전체 업로드 사진`에 저장

2. 반 전체 영상 업로드
→ 선생님 앨범의 `반 전체 영상 업로드`에 저장

3. 학생 사진 업로드
→ 선생님 앨범의 `개별 사진 업로드`에 저장

4. 학생 영상 업로드
→ 선생님 앨범의 `개별 영상 업로드`에 저장

단, 폴더를 학생별로 복잡하게 나누지는 말고,
각 미디어에 아래 메타 정보만 표시되게 해줘.

- 어느 일지에서 올라온 것인지
- 반 전체 업로드인지 / 개별 업로드인지
- 날짜
- 관련 반명 또는 학생명

즉 선생님 앨범에서는 폴더를 복잡하게 쪼개지 말고
미디어 카드 또는 리스트에 `경로/출처`만 표시하면 됨.

예:
- 2026-03-21 / 월 13:00반 / 반 전체 사진
- 2026-03-21 / 서태웅 / 개별 사진

[4. 학부모 일지 화면 노출 규칙]

일지를 저장하고 나면 연결된 학부모 모드에서 그 미디어를 볼 수 있게 해줘.

학부모는 일지 상세 화면에서
- 일지 본문 아래
- 사진/영상이 세로로 쭉 이어져서 보이게 해줘
- 스크롤 내리면서 순서대로 보게 해줘
- 순서는 선생님이 올린 순서 그대로

[5. 학부모 다운로드 규칙]

학부모 일지 화면에서는 미디어를 보는 건 가능하고,
다운로드는 각 미디어를 눌렀을 때 가능하게 해줘.

1. 사진을 눌렀을 때
- 사진 상세 보기 열기
- 이 화면에 버튼 2개 표시
  - 이 사진 다운로드
  - 전체 다운로드

2. 영상이 있을 때
- 실시간 재생은 하지 말 것
- 영상 썸네일만 먼저 보여주기
- 썸네일 또는 영상 카드 눌렀을 때 아래 문구 표시
  - `영상은 다운로드 후 볼 수 있습니다`
- 버튼 2개 표시
  - 이 영상 다운로드
  - 전체 다운로드

즉 영상은 앱 안에서 바로 재생하지 말고 다운로드 기반으로 처리

[6. 학부모별 노출 분기 규칙 (가장 중요)]

반 전체 사진/영상과 특정 학생 개별 사진/영상을 섞어서 올릴 수 있으므로
학부모에게는 아래 기준으로 다르게 보여줘야 함.

예:
- 반 전체 사진 4장 업로드
- 특정 학생(서태웅)에게만 개별 사진 2장 추가 업로드

그 결과:
1. 서태웅 학부모
→ 총 6장 보임
(반 전체 4장 + 서태웅 개별 2장)

2. 다른 학생 학부모
→ 총 4장만 보임
(반 전체 4장만)

즉,
- 반 전체 업로드는 해당 반 학부모 모두에게 노출
- 개별 업로드는 해당 학생 학부모에게만 노출

이 규칙을 사진/영상 둘 다 동일하게 적용해줘.

[7. 학부모 앨범 저장 규칙]

학부모 앨범에도 선생님이 올린 사진/영상이 자동으로 쌓이게 해줘.

단, 학부모 앨범은 복잡하게 분류하지 말고 아래 기준으로 단순하게 간다.

1. 반 전체 사진 / 개별 사진 구분 없이 한 앨범에 모두 모아 보여줌
2. 반 전체 영상 / 개별 영상도 별도 세부 분류 없이 영상 항목으로 함께 모아 보여줌
3. 단, 각 미디어마다 `어느 경로에서 온 것인지`는 표시

예:
- 월 13:00반 일지에서 온 사진
- 서태웅 개별 일지에서 온 사진
- 반 전체 영상 업로드
- 개별 영상 업로드

즉 학부모는 분류보다 “내가 받은 것 전체 보기”에 집중하게 해줘

[8. 일지/앨범 UI 표시 방식]

1. 사진은 세로 나열
2. 업로드 순서 유지
3. 영상은 썸네일 카드 + 다운로드 안내 문구
4. 사진/영상이 섞여 있어도 올린 순서대로 나열
5. 학부모 화면에서는 `반 전체/개별`을 굳이 강조하지 말고 자연스럽게 보여줘
6. 다만 내부 필터링은 정확히 해서 각 학부모에게 보이는 수량이 달라야 함

[9. 확인해야 할 테스트]

작업 후 아래를 직접 확인해줘.

1. 반 공통 일지에 사진/영상 업로드 버튼이 생겼는지
2. 학생별 추가 일지에 사진/영상 업로드 버튼이 각각 생겼는지
3. 업로드 후 일지 작성 화면 아래에 업로드 순서대로 미리보기가 보이는지
4. 선생님 앨범에서 반 전체 사진 / 반 전체 영상 / 개별 사진 / 개별 영상이 각각 저장되는지
5. 학부모 일지 화면에서 본문 아래로 사진/영상이 세로 스크롤로 보이는지
6. 사진 클릭 시 `이 사진 다운로드 / 전체 다운로드`가 나오는지
7. 영상 클릭 시 `영상은 다운로드 후 볼 수 있습니다` 안내와 다운로드 버튼이 나오는지
8. 반 전체 4장 + 특정 학생 2장 업로드했을 때
   - 해당 학생 학부모는 6장 보이는지
   - 다른 학생 학부모는 4장만 보이는지
9. 학부모 앨범에 받은 사진/영상이 모두 누적되는지
10. 학부모 앨범에서 각 미디어 출처(어느 일지/어느 경로)가 표시되는지

[10. 최종 목표]

- 선생님은 일지 작성 중 반 전체/학생별로 사진과 영상을 자유롭게 업로드할 수 있어야 함
- 학부모는 자기 자녀에게 해당되는 사진/영상만 자연스럽게 전달받아야 함
- 사진/영상은 일지와 앨범 모두 연결되되,
  학부모에게는 복잡한 분류 없이 `내가 받은 것 전체`처럼 보이게 해줘
