[스윔노트 Expo Go 스플래시 이미지 교체 지시서]

지금 iPhone에서 Expo Go로 실행할 때 보이는 기본 이미지(현재 보이는 A 모양 기본 이미지)를
기존 Expo 기본 이미지가 아니라
내가 이미 업로드해 둔 “스윔노트 로고 이미지”로 교체해라.

중요:
- 지금 변경 대상은 앱 내부 화면이 아니라 “앱 실행 직후 보이는 스플래시/런치 화면 이미지”다
- 현재 기본 Expo/기본 샘플 이미지가 계속 나오고 있으니 완전히 제거하고 교체해야 한다
- 이미 업로드된 로고 이미지를 사용해라
- 다른 임시 이미지나 기본 예시 이미지 사용 금지
- 기존 이미지 캐시 때문에 안 바뀌는 경우까지 고려해서 확인해라
- 실제 iPhone Expo Go에서 다시 실행해서 변경 반영 여부까지 확인하고 보고해라

==================================================
1. 작업 목표
==================================================

- Expo Go 실행 시 처음 보이는 스플래시 이미지를
  내가 업로드해 둔 스윔노트 로고 이미지로 변경
- 기존 기본 이미지 완전 제거
- iOS 기준 실제 실행 확인
- Android 설정도 함께 점검해서 동일 로고가 적용될 수 있게 정리

==================================================
2. 사용 이미지
==================================================

- 이미 업로드된 로고 이미지 파일을 사용
- 새 이미지 생성 금지
- 임의 대체 이미지 사용 금지

중요:
내가 업로드한 로고 이미지의 경로를 정확히 찾아서
스플래시 설정에 연결해라.

만약 현재 업로드된 이미지가
- 해상도가 너무 낮거나
- 비율이 깨지거나
- 스플래시에 쓰기 어려운 품질이면

그 경우에는
“현재 업로드된 이미지 품질이 부족하여 재업로드가 필요합니다”
라고 보고해라.

하지만 먼저 지금 업로드된 이미지로 최대한 적용 시도부터 해라.

==================================================
3. 수정 대상
==================================================

아래를 전부 점검하고 수정해라.

- app.json 또는 app.config.js / app.config.ts 의 splash 설정
- expo-router / expo 환경에서 스플래시 관련 설정
- iOS용 splash/image 경로
- Android용 splash/image 경로
- backgroundColor
- resizeMode (contain / cover 여부 적절히 설정)

우선 목표:
- iPhone Expo Go에서 기본 이미지가 안 나오고
- 스윔노트 로고가 보이게 만들 것

==================================================
4. 설정 규칙
==================================================

- splash.image 에 업로드된 로고 이미지 연결
- 배경색은 현재 앱 성격에 맞게 단순하고 깔끔하게 설정
- 로고가 잘리지 않게 할 것
- 로고가 너무 작거나 너무 크게 깨지지 않게 조정
- 기본 Expo 샘플 이미지 참조 제거

권장:
- resizeMode는 contain 우선 검토
- 배경색은 흰색 또는 현재 로고와 가장 자연스러운 단색으로 맞춰라

==================================================
5. 캐시/반영 확인 (중요)
==================================================

스플래시 이미지는 캐시 때문에 안 바뀌는 경우가 많으니
반드시 아래까지 확인해라.

- Expo 캐시 초기화
- 다시 빌드/재시작
- iPhone Expo Go 재실행
- 실제로 첫 화면 이미지가 바뀌었는지 확인

필요 시:
- expo start -c 수준의 캐시 초기화도 검토
- iOS에서 다시 실행해서 반영 확인

==================================================
6. 확인해야 할 것
==================================================

반드시 실제 확인:

1. 기존 기본 A 모양 이미지가 사라졌는지
2. 업로드한 스윔노트 로고가 스플래시에 표시되는지
3. 이미지가 찌그러지지 않는지
4. 배경색이 어색하지 않은지
5. iPhone Expo Go에서 실제 반영되는지

==================================================
7. 완료 보고 형식
==================================================

작업 후 아래 형식으로 보고해라.

1. 스플래시에 연결한 실제 이미지 파일 경로
2. 수정한 설정 파일 목록
3. 적용한 배경색 / resizeMode
4. 캐시 초기화 여부
5. iPhone Expo Go 실제 확인 결과
6. 현재 업로드된 이미지 품질이 충분한지 여부
7. 품질이 부족하면 재업로드 필요 여부

중요:
실제로 반영 확인 안 했으면 완료라고 하지 마라.
“설정만 바꿨다”로 끝내지 말고
iPhone Expo Go에서 첫 실행 화면이 바뀌었는지 반드시 확인하고 보고해라.
