2. 관리자 홈 — 운영 현황 카드
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[카드 순서]

1. 이번 달 매출  
2. 전체 회원 수  
3. 보강 상태  
   - 남은 보강  
   - 미처리 보강  
4. 통합 사용량 카드  

---

[통합 사용량 카드 구성]

한 카드 안에 3지표 포함:

① 회원 사용량  
- 현재회원 / 최대회원  
- 구독 기준 자동 계산  

② 데이터 사용량 (사진 포함)  
- % 표시  

③ 영상 사용량  
- % 표시 (별도 저장소)

---

[데이터 기준]

- 무료: 10명 / 기본 용량  
- 유료: 인원 증가 + 용량 증가  
- 사진: 구독 포함  
- 영상: 별도 과금  

---
