슈퍼관리자 모드를 처음부터 끝까지 아래 기준으로 새로 정리해서 구현해줘.
이번 작업은 다른 모드와 완전 연동이 아직 안 되어도 괜찮으니, 우선 단독으로 동작하는 구조를 먼저 만들어줘.

[1. 슈퍼관리자 기본 구조]
- 하단 탭 제거
- 아이콘형 홈으로 구성
- 홈 아이콘 6개:
  1) 운영자 관리
  2) 구독 관리
  3) 저장공간 관리
  4) 킬스위치
  5) 정책 관리
  6) 운영 로그

[2. 운영자 관리]
- 수영장/1인 코치/레슨팀 등 회원 운영 주체를 모두 관리 대상으로 처리
- 각 운영자 카드에 아래 표시:
  - 운영자명
  - 상태(승인 대기 / 운영 중 / 제한 / 종료)
  - 활성 회원 수
  - 현재 구독
  - 저장공간 사용량
  - 마지막 접속일

[3. 회원 상태 정책]
- 회원 상태는 정상 / 연기 / 퇴원 3개만 사용
- 활성 회원 수 = 정상 + 연기
- 퇴원은 제외

[4. 구독 관리]
- 자동 구독 변경 금지
- 운영자가 직접 구독 변경
- 즉시 적용
- 일할 계산 적용
- 업그레이드: 차액 즉시 결제
- 다운그레이드: 차액 credit_balance 적립
- 결제일은 변경하지 않음

구독 관리 화면에 표시:
- 현재 플랜
- 현재 활성 회원 수
- 추천 플랜
- 다음 결제일
- 현재 크레딧
- 구독 변경 버튼

[5. 크레딧 정책]
- 다운그레이드 차액은 환불하지 않고 크레딧 적립
- 다음 결제에서 자동 차감
- 다음 결제가 오기 전에 해지하면 남은 크레딧 환불 가능

[6. 저장공간 관리]
- 자동 용량 업그레이드 금지
- 운영자가 직접 용량 변경
- 총 용량 = 기본 용량 + 추가 용량
- 저장공간 95% 이상이면 신규 업로드 차단
- 조회/삭제는 가능

저장공간 관리 화면에 표시:
- 현재 사용량
- 기본 용량
- 추가 용량
- 총 용량
- 95% 경고 여부
- 용량 변경 버튼
- 파일 삭제 버튼

[7. 구독 해지 정책]
- 해지 즉시 read-only 전환
- 업로드 차단
- 다운로드 차단
- 사진/영상은 24시간 후 자동 삭제
- 복구 불가
- 텍스트 데이터는 유지

[8. read-only 해제 조건]
- 활성 회원 수(정상 + 연기)가 30명 이하가 되면 자동 해제

[9. 킬스위치]
삭제 방식 3개 제공:
- 전체 삭제
- 기간별 삭제
- 항목별 삭제

기간 옵션:
- 3개월
- 6개월
- 1년
- 직접 입력

항목 옵션:
- 사진
- 영상
- 수업일지
- 출결
- 보강
- 메모
- 쪽지

실행 전 반드시 표시:
- 삭제 대상
- 삭제 기간
- 삭제 항목
- 예상 삭제량
- 복구 불가 안내

[10. 정책 관리]
- 정책 문서 작성/수정/저장 기능
- 우선 `환불 정책` 페이지부터 생성
- 슈퍼관리자 모드에서 작성 가능
- 운영자 설정 화면에서는 읽기 전용으로 볼 수 있게 연결

[11. 환불 정책 페이지 기본 문구 저장]
아래 문구를 기본값으로 넣어줘.

"구독 변경은 즉시 적용됩니다.
상위 플랜 변경 시 남은 기간 기준 차액이 즉시 결제됩니다.
하위 플랜 변경 시 남은 기간 기준 차액은 환불되지 않고, 다음 결제 시 차감되는 크레딧으로 적립됩니다.
구독 해지 시 유료 기능은 즉시 제한되며, 서비스는 읽기전용 상태로 전환됩니다.
구독 해지 후 24시간이 경과하면 저장된 사진 및 영상 데이터는 자동 삭제되며 복구되지 않습니다.
이미 결제된 이용요금은 원칙적으로 환불되지 않습니다.
단, 다음 결제가 발생하지 않는 상태에서 남아 있는 크레딧은 환불될 수 있습니다.
사용자는 구독 해지 전 데이터 삭제 정책을 충분히 확인해야 하며, 삭제된 데이터는 복구되지 않습니다."

[12. 운영 로그]
아래 로그를 기록/조회 가능하게 해줘.
- 운영자 승인/반려
- 구독 변경
- 크레딧 적립
- 용량 변경
- 킬스위치 실행
- 정책 수정

로그 항목:
- 시간
- 작업자
- 대상
- 작업 내용

[13. UI 규칙]
- 하단 탭 없음
- 아이콘형 홈
- 홈 버튼/뒤로가기/팝업 동작은 다른 모드와 통일
- 팝업은 빈 화면 클릭/X 버튼으로 닫힘

[14. 현재 우선순위]
- 다른 모드와 완전 연동되지 않아도 괜찮음
- 우선 슈퍼관리자 화면/구조/상태값/정책 저장 기능부터 단독으로 동작하게 만들 것
