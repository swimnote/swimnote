선생님 앱 하단탭에 "오늘 스케쥴" 탭을 다시 추가해줘.

목표:
1. 현재 사라진 "오늘 스케쥴" 탭을 복구
2. 선생님이 오늘 담당하는 수업을 한눈에 볼 수 있게 카드형 레이아웃으로 표시
3. 오늘 해야 할 일이 헷갈리지 않게 시간순으로 정리
4. 내반 / 출결 / 수영일지 / 사진영상 데이터와 자동 연동
5. 관리자 모드에서도 같은 수업 스케쥴 데이터가 연동되게 해줘
6. 오늘 스케쥴 안에서는 수업 자체를 수정할 수 없고 선생님 개인메모만 추가 가능하게 해줘
7. 개인메모는 텍스트 메모 + 음성녹음 메모 둘 다 가능하게 해줘
8. 지난 스케쥴 메모를 볼 수 있는 별도 카드도 탭 안에 추가해줘
9. 지난 스케쥴 메모 카드 클릭 시 달력이 열리고 메모가 있는 날짜는 체크 표시되게 해줘
10. 날짜를 누르면 해당 날짜의 메모와 음성녹음 내용을 깔끔한 화면으로 볼 수 있게 해줘
11. 선생님 개인메모는 기본적으로 선생님만 조회 가능하게 하고 학부모와는 연동하지 않게 해줘
12. 관리자도 수업 스케쥴은 볼 수 있지만 선생님 개인메모는 기본 비공개로 처리해줘

핵심 구조:
- 오늘 스케쥴 데이터는 기존 class_groups / students / attendance / swim_diary 데이터를 조합해서 자동 생성
- 오늘 날짜 기준으로 teacher_user_id와 schedule_days를 보고 해당 선생님의 오늘 수업만 시간순으로 조회
- 오늘 스케쥴은 수정 불가, 메모만 작성 가능
- 메모는 수업일지와 분리된 별도 테이블로 저장

새 PostgreSQL 테이블 생성:
teacher_schedule_notes

필드:
- id
- teacher_id
- class_group_id
- swimming_pool_id
- schedule_date
- note_text
- audio_file_url (optional)
- created_at
- updated_at

2. PostgreSQL 테이블 생성

10. 음성메모 파일은 사진/영상과 동일한 영구 저장 방식(Object Storage 등)으로 저장
11. 앱 재실행 후에도 메모와 음성파일이 유지되게 해줘

UI 요구:
- 선생님 하단탭:
  내반 / 오늘 스케쥴 / 출결 / 수영일지 / 사진·영상
- "오늘 스케쥴" 탭 메인 상단:
  오늘 날짜
  오늘 총 수업 개수
  오늘 해야 할 일 요약
- 아래에는 시간순 카드 리스트
- 각 카드에는:
  수업시간
  반 이름
  학생 수
  출결 상태
  수영일지 작성 여부
  빠른 이동 버튼(출결 / 일지)
  개인메모 버튼
- 카드 레이아웃은 깔끔하고 실수 없게 큰 텍스트와 상태배지 중심으로 정리

오늘 스케쥴 카드 예시 표시 항목:
- 15:00
- 초급반
- 학생 5명
- 출결 미완료 / 완료
- 일지 미작성 / 작성완료
- 신규 학생 여부
- 보강 여부가 있으면 작은 배지

개인메모 기능:
- 카드 안에서 "개인메모" 버튼 클릭
- 바텀시트 또는 모달 열기
- 텍스트 입력 가능
- 음성녹음 시작 / 종료 버튼 제공
- 저장 시 teacher_schedule_notes에 저장
- 같은 날짜, 같은 class_group_id 기준으로 기존 메모가 있으면 update

지난 스케쥴 메모 기능:
- 오늘 스케쥴 탭 상단 또는 하단에 "지난 스케쥴 메모" 카드 추가
- 카드 누르면 달력 화면 열기
- 메모가 있는 날짜는 점 또는 체크 표시
- 날짜 선택 시 메모 상세 화면으로 이동
- 메모 상세 화면에는:
  날짜
  반 이름
  텍스트 메모
  음성메모 재생 버튼
  작성일 표시

관리자 연동:
- 관리자도 같은 수업 스케쥴 데이터를 확인 가능
- 단, teacher_schedule_notes의 개인메모는 기본적으로 관리자에게도 비공개
- 필요하면 나중에 권한 확장 가능하게 설계만 해두고 지금은 teacher_id 본인만 조회 가능하게 해줘

학부모 연동:
- 학부모는 선생님 개인메모를 보지 않음
- 학부모는 기존 수영일지 / 출결 / 사진영상만 연동
- 개인메모는 절대 노출되지 않게 API 레벨에서 차단해줘

구현 방식:
- Expo React Native + TypeScript
- Expo Router 구조 유지
- 오늘 스케쥴 화면 파일 새로 생성 또는 복구
- 공통 카드 컴포넌트로 분리
- 상단 고정 / 하단 목록 스크롤 구조
- 목록은 FlatList 사용
- 음성녹음은 Expo에서 사용 가능한 안정적인 녹음 방식으로 구현
- 음성파일 업로드 후 audio_file_url 저장
- 앱 재실행 후에도 다시 조회 가능하게 연결

추가 점검:
1. 내반 / 출결 / 오늘 스케쥴 / 수영일지 간 데이터 연동 확인
2. 오늘 스케쥴 카드에서 출결 버튼 누르면 해당 반 출결 화면으로 이동
3. 오늘 스케쥴 카드에서 일지 버튼 누르면 해당 반 일지 작성 화면으로 이동
4. 일지 작성 완료 후 오늘 스케쥴 카드 상태가 바로 갱신되게 해줘
5. 출결 완료 후 오늘 스케쥴 카드 상태가 바로 갱신되게 해줘

최종 결과:
- 선생님이 오늘 해야 할 수업을 한눈에 파악 가능
- 메모와 음성메모로 수업 준비 가능
- 지난 메모도 날짜별로 다시 확인 가능
- 운영 데이터와 개인메모가 섞이지 않게 분리
- 관리자/선생님/학부모 권한 구조도 유지
